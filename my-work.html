Weather App

This weather app allows users to search for any city and view real-time weather information, including temperature, weather conditions, humidity, and wind speed. It was built using HTML, CSS, and JavaScript and fetches live data from the SheCodes weather API. The project is version-controlled with GitHub and deployed online using Netlify.

City Time Checker

A simple web application that displays the real-time date and time for a few selected cities around the world. Built using HTML, CSS, JavaScript, and the Moment.js library, it provides accurate time updates through a clean and responsive interface. The project emphasizes clarity and ease of use, offering a quick way to check time zones in specific locations. Version control was managed with GitHub, and the application is hosted on Netlify for fast and reliable access across devices.
